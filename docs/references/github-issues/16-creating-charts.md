# Creating Charts

> Source: https://docs.github.com/en/issues/planning-and-tracking-with-projects/viewing-insights-from-your-project/creating-charts

## Overview

This guide explains how to create new charts to save your configurations in GitHub Projects.

## Prerequisites

- Access to a GitHub project
- Appropriate permissions to create and configure insights

## Step-by-Step Instructions

### 1. Navigate to Your Project

Begin by going to your GitHub project.

### 2. Access the Insights Feature

In the top-right corner, click the insights button (graph icon) to enter the insights view.

### 3. Create a New Chart

In the left sidebar menu, select **New chart**.

### 4. Name Your Chart (Optional)

To customize the chart name:
- Click the dropdown arrow icon next to the chart name
- Enter your desired name
- Press **Return** to confirm

### 5. Apply Filters

Above the chart, add filters to customize the data displayed. See the Filtering projects documentation for detailed filtering instructions.

### 6. Save Your Configuration

To the right of the filter text box, click **Save changes** to preserve your chart settings.
